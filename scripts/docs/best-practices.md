# Best Practices

## Security
- Use IAM roles with least privilege
- Avoid hardcoding credentials
- Use AWS regions where Bedrock is supported

## Cost Control
- Limit max_tokens
- Monitor usage with CloudWatch
- Use smaller models for PoCs

## Prompt Engineering
- Keep prompts concise
- Use clear instructions
- Add guardrails for unknown answers

## Development
- Start simple
- Validate with notebooks
- Move to production gradually
